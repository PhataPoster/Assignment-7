# Keen Keeper

Keen Keeper is a friendship interaction tracking web app where users can manage friend activity and view timeline-based communication history.

## Short Description

This project helps users track social interactions (call, text, and video call) with friends in a clean dashboard-style interface. It includes timeline filtering and visual analytics for better insight into communication patterns.

## Technologies Used

- Next.js (App Router)
- React
- Tailwind CSS
- Recharts
- JavaScript (ES6+)

## Key Features

- Interaction Timeline: Shows friend activity records with communication type (Call, Text, Video Call).
- Actionable Filter: Filter timeline entries by interaction type using a select input.
- Analytics View: Displays friendship interaction distribution with a pie chart.

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.
